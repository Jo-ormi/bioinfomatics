miRNA로 검색

EX) hsa-miR-23c, hsa-miR-31-5p, ...


target gene으로 검색

  • EX) Gene ID -> 29974, 1, ....
  • EX) Ensembl ID -> ENSG00000148584, ....
  • EX) OMIM number-> 618199, 103950, ...
  • EX) Gene name-> A1CF, LMBR1, NT-3,...



출처 데이터베이스
  • TargetScanHuman
    Human에 대한 miRNA targets을 예측하는 프로그램
  • TarBase
    동물실험으로 얻은 microRNA target의 포괄적인 데이터베이스
  • miRecords
    microRNA-target interaction을 위한 통합 리소스
  • miRDB
    miRNA target prediction과 functional annotations을 제공하는 온라인 데이터베이스
  • PITA
    microRNA target 인식에서 mRNA 내의 base-pairing interactions에 의해 결정되는 target-site 접근성의 역할을 통합하는 데이터베이스


target gene 종류
  • Gene id
    해당 유기체의 특정 위치에 대한 안정적인 ID
  • Ensembl id
    ENS로 시작하는 gene에 대한 unique한 ID
  • OMIM(Online Mendelian Inheritance in Man) number
    질병, 유전자 또는 표현형에 대한 코드
